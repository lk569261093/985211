# 智学伴侣 - 中学生智能探究系统

## 简介

智学伴侣是一个基于 Streamlit 开发的中学生智能学习辅助系统，利用大语言模型为学生提供个性化的学习辅导服务。

## 功能特性

- 💬 **学习对话**：智能问答，覆盖小学到高中各学科知识点
- 📖 **学科导航**：按学科和学段浏览知识点
- 🧰 **学习工具箱**：
  - 学习计划生成
  - 练习题生成
  - 知识点卡片
- ✍️ **作文写作**：
  - 语文作文生成（支持提纲/范文）
  - 英语作文生成
  - 高考作文主题预测
- 📚 **学习档案**：自动保存学习记录

## 部署说明

### 环境要求

- Python 3.10+
- Streamlit
- OpenAI Python 客户端

### 安装依赖

```bash
pip install -r requirements.txt
```

### 配置 API

应用不再依赖环境变量，API 通过侧边栏 **“API 设置”** 完成授权配置（用户无需填写 API Key、Base URL 或模型）：

- 在侧边栏复制本机 **机器码**，发送给开发者。
- 开发者使用 `api_key_authorizer.py` 生成授权码。
- 将授权码填入侧边栏完成验证（授权码已包含必要接口信息，可选择本机加密保存）。

开发者授权码工具运行方式：

```bash
python api_key_authorizer.py
```

如需指定接口地址与模型，请在授权码工具中填写（留空将使用 config.yaml 预置值）。
授权码支持设置到期时间（精确到分钟），到期后需重新申请。


如需清除本机保存的授权信息，可在侧边栏点击“清除本机API配置”。

### 运行应用

```bash
streamlit run app.py
```

应用将在浏览器中自动打开（默认地址：http://localhost:8501）。

## 配置文件

`config.yaml` 包含系统的所有配置参数：

- `app_settings`: 应用基本设置
- `ui_settings`: 界面样式设置
- `scenarios`: 学习场景与温度参数
- `subject_settings`: 学科与学段设置
- `subject_library`: 知识点库
- `writing_settings`: 作文写作设置
- `safety_guardrails`: 安全围栏关键词

## 安全说明

- 授权码在应用内配置，支持本机加密保存（与机器码绑定）
- 系统内置安全围栏，过滤非学习内容
- 学习记录自动保存到本地目录
